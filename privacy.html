Privacy Policy

Last Updated: January 2, 2026

"Enhancer 4 Google" (hereinafter referred to as "this extension") prioritizes user privacy. This policy explains how we handle your data in compliance with the Google API Services User Data Policy.

Summary: We do not collect your personal information, browsing history, or input data on our servers. All processing occurs locally within your browser or via direct communication with Google's official APIs.

1. Data Accessed

This extension accesses the following Google user data only when explicitly initiated by the user:

  • Google Drive Files (drive.file): Access is limited ONLY to files created by this extension itself (e.g., temporary image files converted from PDF). We DO NOT access, read, or modify your existing documents or photos in Google Drive.
  • Google Slides (presentations): Access is required to create new slides and insert images into the active presentation initiated by the user.
  • User Identity (userinfo.profile/email): Access is used solely to verify the user's session for Google Drive/Slides API operations via Chrome Identity API.

2. Data Usage

The accessed Google user data is used exclusively for the following purpose:

  • PDF to Slides Conversion: To convert a local PDF file selected by the user into Google Slides. The extension temporarily uploads converted images to the user's Google Drive, inserts them into a Google Slide, and then immediately deletes the temporary files.

We do not use Google user data for advertising, surveillance, or any other purposes.

3. Data Sharing

We do not share your Google user data with any third parties.

All data processing (PDF conversion, image generation) occurs locally within your browser. The interaction with Google APIs (Drive/Slides) is a direct communication between your browser and Google servers. No data is sent to the developer's servers or any third-party analytics services.

4. Data Storage & Protection

This extension is designed to be "stateless" regarding sensitive user data.

  • No Remote Storage: We do not store your tokens, files, or slide content on any external servers.
  • Local Storage: Non-sensitive settings (e.g., UI preferences) are stored locally in your browser using Chrome Storage API.
  • Protection: All API communications use standard HTTPS encryption (OAuth 2.0) provided by Google.

5. Data Retention & Deletion

Since we do not store your personal data on our servers, there is no data retention period.

  • Temporary Files: Image files created in Google Drive during the slide generation process are deleted immediately by the extension once the slide insertion is complete.
  • Deletion Request: Users can stop all data access and remove all locally stored settings simply by uninstalling this extension from their Chrome browser. No further action is required to delete data as we do not hold any.

Contact Us

If you have any questions about this Privacy Policy, please contact us via our GitHub repository.

プライバシーポリシー

最終更新日: 2026年1月2日

「Enhancer 4 Google」（以下、本拡張機能）は、ユーザーのプライバシーを最優先に考えて設計されています。本ポリシーでは、本拡張機能が Google API Services User Data Policy に準拠してどのようにデータを扱うかを説明します。

要約: 本拡張機能は、ユーザーの個人情報、閲覧履歴、入力データを開発者のサーバーへ送信・収集することは一切ありません。すべての処理はユーザーのブラウザおよびGoogleの公式APIとの直接通信によって行われます。

1. アクセスするデータ

本拡張機能は、ユーザーが明示的に機能を利用した場合にのみ、以下のGoogleユーザーデータにアクセスします。

  • Googleドライブのファイル (drive.file): 本拡張機能自身が作成したファイル（PDFから変換された一時的な画像ファイルなど）にのみアクセスします。お客様が作成した既存のドキュメントや写真へのアクセス、読み取り、変更は一切行いません。
  • Googleスライド (presentations): ユーザーが開始したスライド作成プロセスにおいて、新しいスライドの作成および画像の挿入を行うためにアクセスします。
  • ユーザー識別情報 (userinfo): Google Drive/Slides APIを操作する際のセッション認証（OAuth2）を行う目的でのみ使用します。

2. データの使用目的

アクセスしたGoogleユーザーデータは、以下の特定の機能を提供するためだけに使用されます。

  • PDFのスライド変換機能: ユーザーが選択したPDFファイルをスライド化するために使用します。変換された画像を一時的にユーザーのGoogleドライブにアップロードし、Googleスライドに挿入した後、即座に削除します。

ユーザーデータを広告、監視、またはその他の目的で使用することはありません。

3. データの共有

本拡張機能が、Googleユーザーデータを第三者と共有することは一切ありません。

すべてのデータ処理（PDF変換、画像生成）はユーザーのブラウザ内でローカルに行われます。Google APIとの通信は、ユーザーのブラウザとGoogleサーバー間で直接行われ、開発者のサーバーや第三者の分析サービスへデータが送信されることはありません。

4. データの保存と保護

本拡張機能は、機密性の高いユーザーデータを保持しない「ステートレス」な設計になっています。

  • 外部保存なし: 認証トークン、ファイル、スライドの内容を外部サーバーに保存することはありません。
  • ローカル保存: UI設定などの機密でない情報は、ブラウザ内の Chrome Storage API を使用してローカルに保存されます。
  • 通信の保護: Google APIとのすべての通信は、Googleが提供する標準のHTTPS暗号化（OAuth 2.0）によって保護されています。

5. データの保持と削除

個人データをサーバーに保存しないため、データの保持期間はありません。

  • 一時ファイル: スライド生成プロセス中にGoogleドライブに作成された画像ファイルは、スライドへの挿入が完了した直後に、拡張機能によって自動的に削除されます。
  • 削除リクエスト: 拡張機能をブラウザからアンインストールするだけで、すべてのデータアクセス権限とローカル設定が削除されます。他に削除が必要なデータは存在しません。
