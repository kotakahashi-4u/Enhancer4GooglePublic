更新履歴 (Changelog)

Enhancer 4 Google の機能追加と修正の履歴です。バージョンをクリックして詳細を確認できます。

v1.2.3 expand_more
  • FIX NotebookLMのスタジオパネル開閉機能の修正
    NotebookLMのスタジオパネル開閉機能において、閉じた状態でスライドやメモを開き、戻ると開いた状態になってしまう不具合を解消しました。
  • FIX Googleチャットの更新メッセージの動作不具合修正
    Googleチャットにおける送信修飾キーの変更において、更新メッセージの場合には送信修飾キーが効かず更新ボタンを押下しないとメッセージが送信できない不具合を解消しました。
v1.2.2 expand_more
  • FIX NotebookLMカテゴリ管理のリスト表示対応（現在は検索機能のみ）
    NotebookLMのフロントページにおいて、グリッド表示からリスト表示切り替えても検索ボックスを維持するよう修正しました。
  • FIX Geminiカテゴリ管理における保存キーの変更
    chrome.storage.syncの仕様上、1つのキーに対して8KBのデータしか保存できないため、チャット履歴の保存エラーが頻発していました。これを改善するため、キーの扱い方を変更しました。ただし、chrome.storage.syncは全体としてのデータ容量が少ないため、今後chrome.storage.localなどの利用も見据えた恒久対応を行う予定です。
  • FIX NotebookLMカテゴリ管理機能の緊急アップデート
    GoogleWorkspace以外のアカウント（@gmail.com）においてカテゴリ管理機能をONにするとすべてのNotebookが非表示になる問題を修正しました。
  • FIX NotebookLM生成回数カウンター機能の自動保存不具合修正
    生成回数カウンターの入力エリアにおいて、入力した情報が保存されない不具合が発生していたため修正しました。
v1.2.1 expand_more
  • NEW NotebookLM 生成回数カウンター
    Studioの各アクションボタンの日次のAI生成可能回数を可視化する機能を実装しました。
  • FIX NotebookLM カテゴリ管理の未割当Notebookの不具合修正
    未割当Notebookの順序が意図せず変更されてしまう問題を解消しました。
  • FIX NotebookLM カテゴリ管理の表示切替不具合修正
    NotebookLM上の順序をデフォルトの「新しい順」から「タイトル」に切り替えた際の表示問題を解消しました。
  • FIX NotebookLM カテゴリ管理のカテゴリ内カードのレイアウト修正
    カテゴリに分類されたカードはコンパクト化し、よりレイアウトを見やすく修正しました。
  • FIX NotebookLM 同期スケジューラーの不具合修正
    多数のソース（10個以上など）を登録した際に、一部しか同期処理が実行されない問題を解消しました。
  • FIX Geminiカテゴリ機能の自動開閉修正
    Geminiのカテゴリ機能において、チャット履歴をドラッグアンドドロップした際の自動展開問題を解消しました。

  • NEW Gemini カテゴリ検索機能

v1.1.0 expand_more

  • FIX 送信修飾キーの種類追加
    送信修飾キーに新たにAlt+Enterを追加しました。
