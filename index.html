プライバシー・ファースト

Enhancer 4 Googleは、ユーザーのデータを「外部サーバーへ送信しない」設計思想で作られています。

storage ローカル処理

設定や一時データは Chrome Storage (Local) にのみ保存されます。開発者が閲覧することは技術的に不可能です。

api 直接通信

スライド作成時は、ブラウザからGoogle API (Drive/Slides) を直接叩きます。中間サーバーを経由しません。

verified_user 安全な権限

必要な権限（storage, sidePanelなど）のみを要求し、無関係なサイトへのアクセス権は取得しません。
